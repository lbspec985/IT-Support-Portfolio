# ServiceDesk Simulator — IT Support Ticket Portfolio

## Overview
This folder contains my hands‑on IT Support ticket projects completed using the **ServiceDesk Simulator**, a realistic training environment designed to mirror real Tier 1 helpdesk workflows.  
Each ticket demonstrates my ability to:

- Troubleshoot technical issues logically  
- Communicate clearly with users  
- Work with tools like Active Directory, Server Room dashboards, Remote Support, and ISP communication channels  
- Identify true root causes (even when multiple issues appear)  
- Document findings in a professional, industry‑standard format  

These tickets represent real‑world scenarios I would encounter in an entry‑level IT Support role.
